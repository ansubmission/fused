# FUSED (Anonymous Review Version)

This repository contains minimal code and examples supporting the
experiments presented in our submission. All sensitive components have been removed, 
and only simplified pseudocode or public data are included.

Contents:
- image/: toy experiments for RGB image retrieval experiments
- sonar/: toy experiments for evaluating sonar retrieval
- figures/: scripts for generating the visualizations in the paper

This version is intended solely for reviewers to evaluate the technical contributions and is not the complete production codebase. After the review process concludes, a full, non-anonymous version of this repository will be made available with complete implementation details and comprehensive documentation.
